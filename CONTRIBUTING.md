# Contributing Guidelines

To maintain consistency in the Mouse Jiggler project, please follow these rules:

- **Naming Conventions:** Use clear and descriptive names for variables and functions (e.g., `moveMouseCursor` instead of `m1`).
- **Clean Code:** Ensure each function has a single responsibility.
- **Error Handling:** Use try-except blocks for system-level mouse interactions to prevent crashes.
- **Documentation:** Add brief comments for complex logic to help AI tools understand the flow.
